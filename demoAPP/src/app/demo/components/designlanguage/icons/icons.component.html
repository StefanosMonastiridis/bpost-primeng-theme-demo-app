PrimeIcons

PrimeIcons are the default icons provided with PrimeNG.

Download

npm install primeicons --save

Primeicons are installed by default with PrimeNG

Getting Started

PrimeIcons use the pi pi-{icon} syntax such as pi pi-check. A standalone icon can be displayed using an element such as i or span

<i class="pi pi-check"></i>
<i class="pi pi-times"></i>

Size

Size of the icons can easily be changed using the font-size classes of PrimeFlex.

<i class="pi pi-check"></i>
<i class="pi pi-check text-2xl" ></i>

Spinning Animation

Special pi-spin class applies infinite rotate to an icon.

<i class="pi pi-spin pi-spinner text-2xl"></i>

BPOST icons

PrimeIcons are complemented by sets of custom icons designed for the needs of BPOST applications. These sets are:

  • Action icons: used to represent UI actions such as show, hide, close etc
  • Brand icons: specifically designed to enhance the overall bpost customer experience, providing a friendly look and feel. Brand icons are further organised in the following categories
    • Locations
    • Delivery status
    • Lockers
    • Parcel and mail
    • Payment and security
    • People

Installation

Action icons are included in the bpost-theme folder as icon fonts. In order to use you will have to import their css files in the <head><head/> section of index.html .

<link id="bpost-action-icons" rel="stylesheet" type="text/css" href="assets/themes/bpost/icons/action/style.css">

Brand icons are included in the bpost-theme folder as SVGs. You can implement them as any image or copy their source code directly in the DOM.

Getting Started

Action icons use the bpost-icon bpost-{icon} syntax such as bpost-icon bpost-person. A standalone icon can be displayed using an element such as i or span

<i class="bpost-icon bpost-person"></i>
<i class="bpost-icon bpost-fingerprint"></i>
