Spacing scale

The BPOST PRIME NG theme uses the native PRIMEFLEX classes to implement a spacing scale from 0.25 to 2rem. The scale is implemented by different predefined class according to its use (margin, padding, gap in flex layouts, etc). Please consult PRIMEFLEX for full documentation
