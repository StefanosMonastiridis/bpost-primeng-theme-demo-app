Getting Started

The goal of this demo app is to showcase the BPOST theme for Prime NG components. It requires Angular CLI. In case CLI is not installed already, use the command below to set it up.

npm install -g @angular/cli

Once CLI is ready in your system, extract the contents of the zip file distribution, cd to the directory, run npm install to install the dependenencies from npm and then execute "ng serve" to run the application in your local environment.

The application should run at http://localhost:4200/

Structure

The demo app is based on Sakai. It consists of 3 main parts; the application layout, layout assets and PrimeNG component theme assets. Layout is placed inside the src/app/layout folder, and the assets are in the src/assets/layout folder.

Using the BPost Prime NG Theme in new projects

In order to use the BPost Theme in new projects, follow these steps

  • Start a new Angular project, by running the following commands. During installation, for style sheets select SCSS as a format.

    ng new bpost-project

    cd bpost-project

    ng serve

  • Clean up the default index.html page to start from a blank page.
  • Install PrimeNG by running
    npm install primeng
  • You are also going to need PrimeFlex utilities and PrimeIcons
    • npm install primeflex
    npm install npm install primeicons --save
  • Once PrimeNG is installed, you will have to copy the BPOST theme files from the demo app to the new project. You can locate the files in the assets folder.
  • Finally you have to link the theme to your new application, along with the neccesary icon and utility files. Copy and paste the following in the styles.scss of your new project:

    @import "assets/themes/bpost/theme.scss";

    @import "../node_modules/primeng/resources/primeng.min.css";

    @import "../node_modules/primeflex/primeflex.scss";

    @import "../node_modules/primeicons/primeicons.css";
